## 3. What Didn't Work (and Fixes Applied)

| Issue | Root Cause | Fix |
|-------|-----------|-----|
| `useSearchParams()` build error on `/login` | Next.js 16 requires Suspense boundary for `useSearchParams()` during static prerendering | Wrapped `LoginForm` in `<Suspense>` with loading fallback |
| FMP `/api/v3/` endpoints returning "Legacy Endpoint" errors | FMP deprecated v3 API for new users (post Aug 2025) | Rewrote entire `fmp/client.ts` to use `/stable/` endpoints with `?symbol=` query params |
| FMP `search` endpoint returning empty `[]` | Stable API uses `search-name` not `search` for name-based search | Changed endpoint to `search-name` |
| FMP technical indicators returning empty | Not available on free tier | Removed `fetchTechnicalIndicator`, use `priceAvg50`/`priceAvg200` from quote endpoint instead |
| FMP SP500 constituents restricted | Not available on free tier | Removed `fetchSP500Constituents` function |
| `idx_theses_lookup` partial index with `NOW()` | Postgres requires IMMUTABLE functions in index predicates | Changed to regular composite index including `expires_at` column |
| Supabase `db push` connection failure | Missing database password / not linked | Ran `npx supabase login` + `npx supabase link --project-ref` |
| `create-next-app` name rejection | npm doesn't allow spaces/capitals in package names | Scaffolded with temp name, moved files into project directory |
| Middleware deprecation warning | Next.js 16 deprecated `middleware` in favor of `proxy` | Non-blocking warning; middleware still functional. Can migrate to proxy later |
